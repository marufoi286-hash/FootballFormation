# Vim 操作一覧チートシート

## 1. モード切り替え (Mode Switching)
Vimには主に以下のモードがあります。**Escキー**ですべて「ノーマルモード」に戻ります。

| キー | 説明 |
| :--- | :--- |
| **Esc** | ノーマルモードに戻る（迷ったらこれを連打） |
| `i` | カーソルの**前**から入力開始 (Insert) |
| `a` | カーソルの**後**から入力開始 (Append) |
| `I` | 行頭から入力開始 |
| `A` | 行末から入力開始 |
| `o` | カーソルの**下**に空行を挿入して入力 |
| `O` | カーソルの**上**に空行を挿入して入力 |
| `v` | 文字単位のビジュアルモード (範囲選択) |
| `V` | 行単位のビジュアルモード |
| `Ctrl` + `v` | 矩形（ブロック）選択ビジュアルモード |
| `:` | コマンドラインモードへ移行 |

---

## 2. 保存・終了 (Save & Quit)
※ ノーマルモードで `:` を押してから入力します。

| コマンド | 説明 |
| :--- | :--- |
| `:w` | 上書き保存 (Write) |
| `:q` | 終了 (Quit) |
| `:wq` | 保存して終了 |
| `:q!` | 保存せずに強制終了 |
| `:x` | 保存して終了 (変更がある場合のみ保存) |

---

## 3. カーソル移動 (Cursor Movement)
矢印キーも使えますが、以下のキーを使うとホームポジションから手を離さずに移動できます。

| キー | 説明 |
| :--- | :--- |
| `h` | 左へ移動 |
| `j` | 下へ移動 |
| `k` | 上へ移動 |
| `l` | 右へ移動 |
| `w` | 次の単語の先頭へ移動 |
| `b` | 前の単語の先頭へ移動 |
| `0` (ゼロ) | 行頭へ移動 |
| `^` | 行頭の文字へ移動（インデント無視） |
| `$` | 行末へ移動 |
| `gg` | ファイルの先頭へ移動 |
| `G` | ファイルの末尾へ移動 |
| `Ctrl` + `f` | 1画面分進む (Forward) |
| `Ctrl` + `b` | 1画面分戻る (Back) |

---

## 4. 編集操作 (Editing)

| キー | 説明 |
| :--- | :--- |
| `x` | カーソル下の1文字を削除 |
| `dd` | カレント行を削除（切り取り） |
| `yy` | カレント行をコピー (Yank) |
| `p` | カーソルの後ろ/下にペースト (Put) |
| `P` | カーソルの前/上にペースト |
| `dw` | カーソル位置から単語末まで削除 |
| `cw` | 単語を削除してインサートモードへ (Change Word) |
| `r` | カーソル下の1文字を置換 |
| `u` | 直前の操作を取り消す (Undo) |
| `Ctrl` + `r` | 取り消しをやり直す (Redo) |
| `.` (ドット) | 直前の編集操作を繰り返す |

---

## 5. 検索・置換 (Search & Replace)

| キー/コマンド | 説明 |
| :--- | :--- |
| `/キーワード` | 下方向へ検索 |
| `?キーワード` | 上方向へ検索 |
| `n` | 次の検索候補へ移動 |
| `N` | 前の検索候補へ移動 |
| `*` | カーソル下の単語を下方向へ検索 |
| `:%s/A/B/g` | ファイル全体の「A」を「B」にすべて置換 |
| `:%s/A/B/gc` | 確認しながらすべて置換 |

---

## 6. ウィンドウ分割・タブ (Window & Tab)

| キー/コマンド | 説明 |
| :--- | :--- |
| `:sp` | 画面を上下に分割 (Split) |
| `:vsp` | 画面を左右に分割 (Vertical Split) |
| `Ctrl` + `w`, `w` | 次のウィンドウへ移動 |
| `Ctrl` + `w`, `矢印(hjkl)` | 指定方向のウィンドウへ移動 |
| `:tabnew` | 新しいタブを開く |
| `gt` | 次のタブへ移動 |
| `gT` | 前のタブへ移動 |
